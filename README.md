# LLM-Agent-paper-daily

本仓库基于 arXiv 每日自动检索与更新，与参考项目类似，新增了 **Agent** 论文类别。

- 自动抓取 arXiv 上 LLM / Multimodal / Diffusion Models / Representation Learning / Agent 相关论文
- 每日定时更新 README 与 `docs/index.md`，可选发布为 GitHub Pages
- 支持每周补充 Papers with Code 代码链接与 HuggingFace 模型链接

> 提示：首次运行 GitHub Actions 后，README 与 `docs/index.md` 会被自动重写为“每日论文表格”，本文件仅作为占位说明。
